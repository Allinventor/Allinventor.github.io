Card-Z

A Board Game Creation Tool

"I wish I could create a card or board game and share it with others." - Ai Worm

What is it?

Card-Z is meant to be a drag and drop tool that will allow one to create a card or board game. It is meant to coninside with an online service that will allow one to share the games they create with others. Card-Z is the tool to create them, while the service is meant to allow you to play the game, share them, and more.

When is is avaliable?

It won't be available until I have a working prototype. Afterwhich I have to create the service for which it is meant to work with. In the end, it would be released once a functional and stable beta version is produced.

What are the goals for this project?

  • Allow users to create a Card Game
  • Allow users to create a Board Game
  • Allow users to play multiplayer locally or accross wifi
  • Allow for public and private games/lobbies
  • Allow for users to play against an AI (or more) regardless of game
  • Allow for custom graphics on the games
  • Allow for customs rules and achivements in a game
  • Create a ranking system for most popular game
  • Allow moetization for custom games
  • Allow for extensions packs to be created for each game (individually or as a group)
  • Allow for cross-platform play
  • Allow for offline use/play
  • Allow for export in various formats for custom printing or production
  • Allow for a physical print-out of each game in the future for those who wish to play classicaly
  • Allow for blockchain integration for special games or users, etc.
