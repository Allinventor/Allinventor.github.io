Project Owl

A mobile focused Content Management System

"I wanted an alternative to Wordpress and Grav. They provide an easy way to create a website, but it's not the way I feel we could do it." - Ai Worm

What is it?

A Content Management System (CMS) allows a user to create a website while being able to edit anything and everything directly within the application or by changing a few lines in a file. A simple and quick way to create and maintain a website for a business, blog or a landing page.

Owl is meant to be a new way to create and maintain a website. It's primary focus is to allow the user to maintain their website by creating new pages or changing them through a mobile device. It is meant to be a mobile friendly CMS that is fast and responsive. It attempts to do so by focusing on content first, and then looks. All while being as SEO (Search Engine Optimization) friendly.

When will it be available?

Project Owl is still underdevlopment as I try to create a system that should allow for everything that is essential in a CMS system. Creating Pages, Creating Backups, and Chaning Style/Theme.

My goal is to create a light and heavy version. The light version would be for smaller business, blogs etc. Any website that doesn't require many things to be functional, and can be easily maintained through a mobile device. The heavy version would be named as another project, which focuses on creating more complex business websites that need user accounts and complex components such as learning lessons.
